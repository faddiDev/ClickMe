# ClickMe

Java Simple Game

<b>Installation:</b>
<ul>
  <li>Make sure java is installed on your machine</li>
  <li>Open cmd.exe</li>
  <li>Target your folder application</li>
  <li>Compile first: javac ClickMe.java</li>
  <li>Run it: java ClickMe</li>
</ul>
